Améliorez votre classement dans les moteurs de recherche avec notre plugin de schéma FAQ ! Créez facilement des rich snippets (données structurées) de questions fréquentes pour vos articles WordPress qui se chargeront rapidement et seront bien structurées pour le SEO. Augmentez la visibilité de votre site et aidez vos clients à trouver rapidement les informations qu'ils recherchent. Essayez notre plugin de schéma FAQ dès maintenant !

= Caractéristiques =

Génère un schéma FAQ pour les articles
Utilise les H2 à H6 en tant que questions
Extrait le contenu entre les H2 à H6 pour en faire les réponses associées
Génère une sortie en JSON-LD au format du schéma FAQ
Affiche le code dans le code HTML de la page

= Installation =

Téléchargez le plugin et décompressez-le.
Chargez le plugin dans votre site WordPress en allant dans le menu "Plugins" > "Ajouter nouveau".
Cliquez sur "Téléverser plugin" et sélectionnez le fichier ZIP téléchargé.
Cliquez sur "Installer maintenant" et attendez la fin de l'installation.
Cliquez sur "Activer le plugin".
= Utilisation =

Le plugin ajoutera automatiquement un schéma FAQ pour chaque article de votre site WordPress. Il n'y a rien de plus à configurer.
